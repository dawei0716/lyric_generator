Data Set used:
Corpus of lyrics of all Kendrick songs from: "Kendrick Lamar" (2009),"Section.80" (2011), "Good Kid: M.A.A.D City" (2012), "To Pimp A Butterfly" (2015), "untitled unmastered." (2016), "DAMN." (2017), "Black Panther" (2018)

Model used:
Recurrent Neural Network using Tensorflow

What is "Temperature"?:
It is a hyperparameter. Lower temperatures generate more predictable text (closer in resemblance to actual lyrics).
Higher temperatures generate more creative text, but at the cost of more mistakes such as spelling mistake.

See Github for details on how it was implemented.
